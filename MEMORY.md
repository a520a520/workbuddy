## ⚡ 三少工作效率特征与协作原则（2026-06-20 新增）

> 以下特征由三少本人确认，任何专家/工作区在派发任务时必须遵守。

### 三少的工作卡住模式（三种）

**模式 A：启动困难**
- 表现：任务确定了，但不知道从哪下手，不自觉想放到明天做
- 原因：任务描述太模糊，大脑接不住
- 解法：**给「下一步具体动作」，不给任务名**

**模式 B：修改恐惧**
- 表现：文案写好了，「感觉需要认真读一下看看要不要改」，然后就放着了
- 原因：「判断」和「修改」两个动作搅在一起，大脑过载
- 解法：**判断和修改分开**。第一步只读不改，标 ⚠️ 即可；修改交给我

**模式 C：耐心型任务抵抗**
- 表现：排版、选图、从十几张照片里选9张、核对类工作——进行不下去
- 原因：这类任务需要持续低刺激细致比对，与三少大脑偏好相反
- 解法：**此类任务全部外包给我**，三少只做最后「确认/点头」

### 协作原则（所有任务必须遵守）

1. **任务拆分原则**：任何任务先拆成「三少做」和「AI做」，三少的部分只到「确认」为止
2. **文案修改两步法**：只读不改标 ⚠️ → 把标了的地方交给我改
3. **选图/排版类任务**：照片放到目录 → 我挑好 → 按发布顺序命名 → 三少只扫一眼确认
4. **每天早上清单**：不给「任务名」，给「下一步动作」（如：打开小红书 → 复制文案 → 选第一张照片 → 点发布）
5. **公众号排版**：我生成 HTML → 三少浏览器打开全选复制 → 粘贴进公众号后台 → 手动插图 → 发布
