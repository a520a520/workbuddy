# QClaw 自动化配置指南

> 在 Mac Mini 上为 QClaw 设置以下自动化任务

---

## 前置准备

### 1. 确认共享文件夹路径
```bash
# Mac Mini 终端执行
ls ~/Library/Mobile\ Documents/com~apple~CloudDocs/Shared-WorkBuddy-QClaw/
```
确保能看到所有文件。

### 2. 安装 Bark 并获取 Key
- App Store 搜索「Bark」安装到 iPhone
- 打开 Bark → 右上角「+」→ 复制推送链接
- 链接格式：`https://api.day.app/xxxxxxxxxxxx/`
- 记录 `xxxxxxxxxxxx` 作为 `BARK_KEY`

### 3. 在 QClaw 中设置这些自动化

在 QClaw 中创建以下自动化任务：

---

## 自动化 1：每日简报（每天上午9:00）

**名称：** 每日任务简报
**时间：** 每天 9:00
**任务内容：**

```
执行以下步骤：

1. 读取共享文件夹中的"统一工作日志.md"
2. 读取"新媒体素材计划/当月_素材制作计划.md"  
3. 读取"外出记录/"目录最新文件
4. 按照"QClaw_每日简报模板.md"格式生成今日简报
5. 保存简报到"QClaw_配置/简报存档/YYYY-MM-DD_简报.md"
6. 提取简报核心摘要（200字内）
7. 通过Bark推送到三少手机：
   curl "https://api.day.app/{BARK_KEY}/【{日期}】今日待办{N}项·逾期{M}项"
```

---

## 自动化 2：逾期检查（每2小时，9:00-21:00）

**名称：** 逾期任务检查
**时间：** 每2小时（9:00, 11:00, 13:00, 15:00, 17:00, 19:00, 21:00）
**任务内容：**

```
执行以下步骤：

1. 读取"统一工作日志.md"
2. 筛选出[ ]标记且截止日期已过的任务
3. 如果有逾期任务：
   - 将任务标记更新为[!] 【逾期】
   - 通过Bark推送给三少：
     curl "https://api.day.app/{BARK_KEY}/⚠️逾期提醒：{任务摘要}"
4. 检查未来2天内到期的关键节点
5. 如有明天/后天到期的关键事项，推送提醒
```

---

## 自动化 3：素材进度追踪（每天下午6:00）

**名称：** 素材制作进度追踪
**时间：** 每天 18:00
**任务内容：**

```
执行以下步骤：

1. 读取"新媒体素材计划/当月_素材制作计划.md"
2. 统计各类型素材的完成率
3. 如果某项进度落后于时间进度（如月过半但完成率<40%），推送提醒
4. 更新素材计划文件底部"QClaw追踪区"
5. 推送素材进度摘要到Bark
```

---

## Bark API 完整用法

```bash
# 基础推送
curl "https://api.day.app/{key}/消息标题/消息内容"

# 带链接（点击打开共享文件夹）
curl "https://api.day.app/{key}/任务提醒/有3项待办?url=file:///path"

# 带声音
curl "https://api.day.app/{key}/紧急提醒?sound=alarm"

# 分组
curl "https://api.day.app/{key}/素材进度?group=QClaw"
```
