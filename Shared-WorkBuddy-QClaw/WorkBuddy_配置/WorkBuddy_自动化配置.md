# WorkBuddy 同步自动化配置

> WorkBuddy端的自动化任务
> 创建于：2026-06-20

## 自动化 1：会话结束时更新共享日志

**名称：** 更新共享工作日志
**触发：** 每次WorkBuddy完成实质性工作后
**任务内容：**

```
执行以下步骤：

1. 读取共享文件夹中的"统一工作日志.md"
2. 将本次会话完成的任务标记为 [x]
3. 将新识别的任务追加到对应区域
4. 更新"最后更新"时间戳
5. 如有内容制作完成，同步更新"新媒体素材计划/当月_素材制作计划.md"
```

## 自动化 2：每天早上推送今日清单

**名称：** 今日工作清单推送
**时间：** 每天 8:55
**任务内容：**

```
执行以下步骤：

1. 读取"统一工作日志.md"
2. 提取今日待办任务
3. 提取未来3天关键节点
4. 生成简洁清单（参考QClaw简报格式）
5. 写入共享文件供QClaw读取
6. QClaw会在9:00发送Bark推送
```
*注：此自动化在WorkBuddy端创建，与QClaw的9:00简报配合*

## 自动化 3：每周一生成周计划

**名称：** 周计划生成
**时间：** 每周一 8:30
**任务内容：**

```
执行以下步骤：

1. 读取"统一工作日志.md"进行中任务
2. 读取"新媒体素材计划/当月_素材制作计划.md"
3. 翻阅冷却区中被遗忘的任务
4. 生成本周工作计划
5. 写入"统一工作日志.md"的"本周重点"区域
6. 提醒三少决定冷却区中哪些继续/放弃
```

## QClaw 远程交互流程

### 三少外出时的工作流

```
三少在手机上：
  ↓
发微信给QClaw（或发Bark消息触发）
  ↓
QClaw解析消息 → 写入共享文件
  ↓
WorkBuddy下次打开时感知
```

### QClaw可处理的指令

| 指令格式 | 示例 | 效果 |
|----------|------|------|
| `记录：{内容}` | 记录：见了始兴果园老板，谈了暑假采摘合作 | 写入外出记录 |
| `任务：{内容}` | 任务：制作始兴采摘产品介绍 | 追加到统一工作日志 |
| `完成：{任务关键词}` | 完成：沙溪月预告 | 标记对应任务为[x] |
| `状态` | 状态 | QClaw返回当前任务总览 |
| `简报` | 简报 | QClaw立即生成当前简报 |
